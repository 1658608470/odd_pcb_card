# 说明 
整体待机耗电（5v状态）：3.5-4毫安 使用的3000毫安时能待机大概以到两个月。  
灌入环氧树脂一定记得密封好接口和屏幕！  
刷入固件请 拉低f引脚（与g引脚短接）然后上电！
